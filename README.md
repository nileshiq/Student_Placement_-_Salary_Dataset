# ElitePath Analytics: Student Placement & Salary Insights

## Project Overview
This project provides an end-to-end data analysis and visualization of student placement outcomes and salary packages (LPA). Using a dataset of 9,000 students, the analysis identifies key drivers behind high-value job offers, including technical skills, internships, and academic performance.

## Dataset Highlights
The analysis is based on `student_placement_salary_elite_v2.csv`, featuring:
- **Academic Metrics**: CGPA, Branch, College Tier.
- **Technical Skills**: Python, DSA, ML, Web Dev, Coding Scores.
- **Experience**: Internships, Projects, Backlogs.
- **Outcomes**: Placement Status, Company Type, Job Role, and Salary (LPA).

## Key Visualizations & Insights

### 1. Placement & Salary Distribution
- **Placement Rate**: ~85.6% success rate.
- **Salary Tiers**: Identified three distinct salary clusters: Entry (40-50 LPA), Mid (70-80 LPA), and Elite (100+ LPA).
- **CGPA Paradox**: While CGPA is necessary for eligibility, technical skills and internships show a stronger correlation with "Elite" salary brackets.

### 2. Skill & Experience Impact
- **Technical Mastery**: High scores in DSA and Python are strong predictors of higher compensation.
- **Internship Value**: A direct positive correlation exists between the number of internships and the starting salary "floor."

### 3. Industry Segmentation
- **Top Tech Advantage**: Salaries in "Top Tech" companies significantly outperform MNCs and Startups.
- **Role Performance**: Data Scientists and Web Developers command the highest average packages (~64.9 LPA).

## Visualizations Included
- `general_analysis.png`: Overview of placement rates and salary trends.
- `skills_internships.png`: Correlation between technical scores and compensation.
- `industry_segmentation.png`: Salary breakdown by job role and company type.

## Technologies Used
- **Language**: Python
- **Libraries**: Pandas, Matplotlib, Seaborn

## Conclusion
The data suggests that for students aiming for elite-tier packages, technical skill development and internship experience are more critical than academic branch or marginal CGPA gains.
